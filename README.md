1- Agriculture Management Dashboard

A modern and responsive Agriculture Management Dashboard built using HTML5, CSS3, Bootstrap 5 and Vanilla JavaScript (ES6).  
This project is designed as a frontend-focused assignment to demonstrate UI/UX design, DOM manipulation and data handling using static JSON data only.

2- Live Demo:
https://samras1996.github.io/Agriculture-Management-Dashboard/

3- Features

- Interactive Dashboard Statistics
- Crops Management (Add, Delete, Search)
- Bar Chart & Area Chart (ApexCharts)
- Static JSON Data (No Backend)
- Fully Responsive Layout
- Modern & Elegant UI Design
- Clean & Modular JavaScript Files
- Collapsible Sidebar Navigation

4- Tech Stack

a- Frontend
- HTML5
- CSS3
- Bootstrap 5
- Vanilla JavaScript (ES6+)

b- Libraries
- ApexCharts (for charts)
- Font Awesome (icons)

c- Data Handling
- Static JSON (Frontend only)

7- Modules Overview

a- Dashboard Statistics
- Dynamically rendered using Vanilla JavaScript
- Data loaded from static JSON objects

b- Charts & Analytics
- Bar Chart: Top Crops Production
- Area Chart: Monthly Yield Comparison
- Implemented using ApexCharts with responsive behavior

c- Crops Management
- Add new crop records
- Delete crop records
- Search/filter crops
- Frontend-only CRUD operations
